# Random Password Generator

Автор: Бобылева Станислава


Описание: GUI-приложение на Python (Tkinter), которое генерирует случайные пароли. Пользователь может настраивать параметры пароля (длину и набор символов), сохранять историю генераций в JSON-файл history.json и загружать её при запуске. Применяется модуль random (не криптографически безопасная генерация для учебных целей; при необходимости можно переключиться на secrets).

Особенности:

Ползунок длины пароля (минимум/максимум задаются в коде, диапазон: 4–64)
Чекбоксы для выбора классов символов: цифры, буквы, спецсимволы
Кнопка "Генерировать"
Таблица истории с ранее сгенерированными паролями
История сохраняется в history.json и загружается при старте
Технологии:

Tkinter (GUI, входит в стандартную поставку Python)
Модуль random (для генерации паролей)

Как работает история

История хранится в формате JSON как массив объектов.
Каждый объект содержит:
password: сгенерированный пароль
length: длина пароля
digits: используется ли набор цифр
letters: используется ли набор букв
specials: используются ли спецсимволы
time: временная метка в формате ГГГГ-ММ-ДД ЧЧ:MM:СС
Пример элемента истории: { "password": "Abc123!@def", "length": 12, "digits": true, "letters": true, "specials": true, "time": "2026-05-05 12:34:56" }

Примеры использования (по возможности тесты)

Пример 1: генерация пароля длиной 12 символов с цифрами и буквами

Установить длину 12
Включить цифры и буквы, отключить спецсимволы
Нажать "Генерировать"
Пароль появится в поле и добавится в историю
Пример 2: генерация пароля, состоящего только из спецсимволов

Отключить цифры и буквы
Включить спецсимволы
Установить необходимую длину
Нажать "Генерировать"
